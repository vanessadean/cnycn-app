# CNYCN Texting Demo App

## Setup

Run the following commands:

`gem install bundler`

`bundle`

`bundle exec rake db:migrate`

`cp .env.example .env`

Then add the username, password and Twilio credentials to the `.env` file

## Run

`ruby app.rb`

## Deployed at

https://cnycn-texting-app.herokuapp.com/
